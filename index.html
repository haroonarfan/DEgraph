    BAB e1@<--> BABA[["
      By Chain Rule:

      df = ∂f/∂x dx + ∂f/∂y dy

      Then:

      P = ∂f/∂x |y

      Q = ∂f/∂y |x

      So:

      ∂P/∂y = ∂Q/∂x

      and in a simply connected domain it is necessary and sufficient condition for P(x,y)dx + Q(x,y)dy to be exact.

      Also, df = 0 ⇒ f(x, y) = const. is the solution.
    "]]
    BABA --> BABAB["To Solve:

      f(x,y) = ∫ P dx = ... + h(y) because ∂h(y)/∂x |y = 0

      Then:

      Take the above f to solve for h(y) using Q = ∂f/∂y |x

      This gives the final f(x,y) = const.
    "]
